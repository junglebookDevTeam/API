# 주문상태 변경 API

## Request (POST) ##
<p>URL: https://api.junglebook.co.kr/wms/orders/confirm/{주문상태}</p>
<p><p>Require header: Authorization {api_key} (api_key는 정글북 개발팀에 발급요청 하시기바랍니다. dev@junglebook.co.kr)</p></p>

<p>* {주문상태} (required): 변경할 주문 상태 값</p>
<p>
	{주문상태} = 
	<ul>
		<li>packing: 상품준비중 -> 패킹완료(송장 데이터 입력), 상품준비중 주문만 상태 변경 가능</li>
	</ul>
</p>

## Request parameters ##

``` js
{
    "orderNo": [ //[Required, array], 상태 변경 대상 주문번호
        1234567890123,
        ...,
        ...
    ],
    "invoiceData": { //[Required, object], 송장 정보
        "1234567890123": { // 주문번호 (송장 데이터 입력 대상 주문번호)
            "parcelCompanyNo": 13, // 택배사 코드(롯데택배: 13, 한진택배: 12)
            "invoices": [ // 송장번호
                "310900000000",
                "310900000001",
                "...",
                ...
            ]
        },
        {...}, ...
    }
}
```


## Response (JSON) ##

### 성공시 (그외 모두 실패) ###

``` js
{
    "confirmOrders": [
        1234567890123,
        ...,
        ...
    ]
}
```
